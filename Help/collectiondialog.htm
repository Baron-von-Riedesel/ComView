Collection Dialog

This dialog may be opened from inside properties dialog if property is a collection or an array of variants..
 

 
A context menu will pop up by right-clicking an item with commands:
  • View: This command will be enabled if item is of type IDispatch or IUnknown. If item is of type IDispatch, it will be shown in a properties dialog, else in an object dialog.
  • Explore Mode: If menu item is checked and currently selected item is of type IDispatch, a properties dialog will show details of this item.
Commands accessible through buttons are:
  • Refresh: will refresh current view. Please note that rereading collections sometimes doesn't work as it should.
  • Put Property: This command will try to update values of this property.

Hints

  • If a grid is shown, items may be edited in place. Click on Value column of item to edit.
  • If items are of type IDispatch content of Value column will be default property of this type (if one exists, else nothing will be shown). This may be changed by right-clicking list header and so opening a context menu which will allow to select another property to show in this column.
View properties options for a detailed description of options concerning collection dialog.
