Installation

To install COMView copy files COMView.exe and COMView.chm to a directory of your hard disk.
 
COMView should run on Windows 95/98/ME systems as well as Windows NT/2000/XP. For Windows 95/NT systems ensure that
  • DCOM package is installed (should be always true for NT). Without that feature you won't be able to create objects on remote machines.
  • HTML help control is installed (HHCtrl.OCX). Without it this help cannot be viewed. Installing Internet Explorer 4.0 or a newer version will install this control as well.
  • COMCTL32.DLL version 4.70+ is installed. That should be the case if Internet Explorer 3.0 or a newer version is installed as well.
