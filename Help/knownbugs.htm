Known Bugs and Restrictions

  • Consistency Check: In some rare cases command Check File Links will report a broken link which in fact is not.

    •  
  • ASM Includes: If a method has more than 30 parameters (including this), MASM would complain about "statement too complex". So in this case - and if size of parameter string is greater than 255 chars - the include generation process comments out that prototype, replacing it by a simple one with one parameter so the vtable offsets are calculated correctly.
  • ASM Includes: The order of STRUCTs in generated ASM include files sometimes may need to be rearranged a bit.
